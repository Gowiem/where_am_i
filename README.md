# WhereAmI

A simple Elixir command-line program that approximates the user's location and uploads it online to Gowiem/whereami.com. This is meant to be run on a Cron schedule each day.

## Dependencies

1. Elixir -- `brew install elixir`

## Running

1. `mix deps.get`
2. `mix escript.build`
3. `./where_am_i`
